# SafeMetrica Risk Share Link QR Poster Copy v1

기준일: 2026-06-09
문서 위치: `docs/sales/SAFEMETRICA_RISK_SHARE_LINK_QR_POSTER_COPY_V1.md`
작업명: `docs: add risk share link qr poster copy v1`

## 1. 목적

이 문서는 `SafeMetrica Risk Share Link`를 현장에 게시할 때 사용할 QR 안내문, 포스터 문구, 근로자 안내 멘트, 현장관리자 설명 문구를 정리하기 위한 영업·운영 기준 문서다.

Risk Share Link는 위험성평가 공유·근로자 확인·의견제출·관리자 검토 기록을 QR 또는 링크로 남기는 진입형 상품이다.

현장 게시물은 근로자가 부담 없이 QR을 찍도록 해야 한다.

```text
짧게
쉽게
겁주지 않게
법적 표현은 최소화
확인과 의견제출 중심
```

## 2. 현장 게시물 기본 원칙

QR 안내문은 아래 원칙을 따른다.

```text
1. 근로자가 3초 안에 목적을 이해해야 한다.
2. 법 위반, 처벌, 과태료, 중대재해 면책 같은 표현을 쓰지 않는다.
3. “의무 완료”가 아니라 “위험요인 확인과 의견제출”로 표현한다.
4. 의견이 없어도 제출할 수 있게 안내한다.
5. 위험하거나 불편한 점이 있으면 사진과 함께 남길 수 있게 안내한다.
```

## 3. 메인 포스터 문구 후보

### 3-1. 기본형

```text
오늘 작업 전 위험요인을 확인해 주세요

QR을 찍고
위험성평가 공유 확인과
현장 의견을 제출할 수 있습니다.

의견이 없으면 “의견 없음”으로 제출해 주세요.
위험하거나 불편한 점이 있으면 사진과 함께 남겨 주세요.
```

### 3-2. 짧은형

```text
위험요인 확인 / 의견제출 QR

오늘 작업 전 위험요인을 확인하고
의견이 있으면 남겨 주세요.
```

### 3-3. 근로자 친화형

```text
오늘 현장에서 위험하거나 불편한 점이 있나요?

QR로 남겨 주세요.

- 위험요인 확인
- 안전조치 확인
- 의견 없음 제출
- 위험제보
- 아차사고
- 개선제안
```

### 3-4. 현장관리자 안내형

```text
위험성평가 공유·참여 기록 QR

근로자가 QR로 위험요인을 확인하고,
의견제출과 사진첨부를 할 수 있습니다.

관리자는 접수함에서
공유확인과 위험제보를 분리해 확인합니다.
```

## 4. 포스터 제목 후보

현장 게시물 제목 후보는 다음과 같다.

```text
오늘 위험요인 확인 QR
위험요인 확인 / 의견제출 QR
위험성평가 공유확인 QR
근로자 의견제출 QR
작업 전 안전확인 QR
현장 위험제보 QR
안전조치 확인 QR
```

우선 추천 제목:

```text
위험요인 확인 / 의견제출 QR
```

이유:

```text
근로자가 이해하기 쉽고,
법적 부담감이 적으며,
확인과 제보 기능을 동시에 설명할 수 있다.
```

## 5. 버튼 문구 후보

근로자 화면 버튼 문구는 아래 기준으로 한다.

```text
위험요인 확인하기
확인 후 제출하기
의견 없음, 확인만 제출
위험 또는 개선의견 제출
사진 첨부하기
제출 완료
```

피해야 할 버튼 문구:

```text
법적 확인 완료
위험성평가 완료
교육 인정 제출
의무 이행 완료
면책자료 제출
```

## 6. 근로자 화면 안내문 후보

### 6-1. 첫 화면 안내

```text
오늘 작업 전 확인할 위험요인입니다.
확인 후 아래 항목을 체크해 주세요.
위험하거나 불편한 점이 있으면 의견을 남길 수 있습니다.
```

### 6-2. 확인 체크 안내

```text
□ 위험성평가 내용을 확인했습니다.
□ 오늘 안전조치와 주의사항을 확인했습니다.
```

### 6-3. 의견 없음 안내

```text
특별한 의견이 없으면 “의견 없음”으로 제출해 주세요.
확인 기록으로 저장됩니다.
```

### 6-4. 위험제보 안내

```text
위험하거나 불편한 점이 있으면 간단히 남겨 주세요.
사진이 있으면 첨부할 수 있습니다.
```

### 6-5. 익명 제출 안내

```text
필요한 경우 익명으로 제출할 수 있습니다.
다만 조치 확인을 위해 위치나 상황은 구체적으로 적어 주세요.
```

## 7. 제출 완료 문구

### 7-1. 공유확인 완료

```text
확인 기록이 제출되었습니다.

오늘 작업 전 위험요인과 안전조치 확인 기록으로 저장됩니다.
```

### 7-2. 위험제보 완료

```text
의견이 접수되었습니다.

현장관리자가 내용을 확인하고
필요한 경우 조치사항을 기록합니다.
```

### 7-3. 사진첨부 완료

```text
사진이 함께 제출되었습니다.

현장 확인과 조치 검토 자료로 활용될 수 있습니다.
```

## 8. 현장관리자 설명 문구

현장관리자가 근로자에게 설명할 때는 아래처럼 말한다.

```text
오늘부터 위험요인 확인과 의견제출은 이 QR로 같이 받겠습니다.

긴 내용 쓰라는 게 아니고,
위험요인 확인했는지 체크하고,
의견이 없으면 의견 없음으로 제출하면 됩니다.

위험하거나 불편한 점이 있으면 사진과 함께 남겨 주세요.
```

짧은 조회 멘트:

```text
작업 전 QR 한 번 찍고 위험요인 확인해 주세요.
의견 없으면 의견 없음으로 제출하면 됩니다.
```

## 9. 대표·담당자 설명 문구

대표나 관리팀에 설명할 때는 아래처럼 말한다.

```text
이 QR은 위험성평가를 대신하는 기능이 아닙니다.

이미 작성된 위험성평가가 근로자에게 공유됐는지,
근로자가 의견을 낼 수 있었는지,
관리자가 검토 기록을 남겼는지를 연결하는 운영지원 도구입니다.
```

영업용 짧은 문구:

```text
위험성평가를 문서 보관에서 끝내지 않고,
근로자 확인·의견제출·관리자 검토 기록으로 연결합니다.
```

## 12. 디자인 구성 기준

QR 게시물은 A4 또는 A3 기준으로 사용한다.

권장 구성:

```text
상단: 제목
중앙: QR 코드
하단: 3단계 안내
맨 아래: 문의 담당자 / 담당 부서
```

3단계 안내 예시:

```text
1. QR 접속
2. 위험요인 확인
3. 의견 없음 또는 위험제보 제출
```

글자 기준:

```text
제목은 크게
설명은 짧게
버튼 문구처럼 표현
법률 용어 최소화
```

## 13. 색상·톤 기준

권장 톤:

```text
깨끗한 흰색 배경
짙은 네이비 또는 검정 제목
QR 주변 충분한 여백
위험 문구는 붉은색 과다 사용 금지
```

이유:

```text
현장 게시물은 눈에 잘 띄어야 하지만,
과도하게 경고성으로 보이면 근로자가 방어적으로 반응할 수 있다.
```

## 14. 금지 표현

QR 포스터와 근로자 화면에는 아래 표현을 쓰지 않는다.

```text
위험성평가 법적 완료
근로자 참여 의무 자동 충족
법정교육 인정 완료
교육시간 인정 보장
과태료 방지
중대재해 면책
법적 방어 보장
무재해 보장
KOSHA 인정 보장
안전조치 완료 확정
QR 제출만으로 법적 의무 완료
```

## 16. 개인정보 안내 문구 후보

근로자 이름 또는 사진을 받을 경우 아래 문구를 화면에 표시할 수 있다.

```text
제출된 내용은 사업장의 위험성평가 공유확인, 현장 의견 검토, 안전조치 기록 관리를 위해 사용됩니다.
사진에는 불필요한 개인정보나 얼굴이 포함되지 않도록 주의해 주세요.
```

익명 제출 안내:

```text
익명 제출을 선택할 수 있습니다.
다만 조치를 위해 위치와 상황은 구체적으로 적어 주세요.
```

```text
토큰, API Key, 환경변수 값이 노출되지 않는가?

담당자 연락처가 최신인가?
```
