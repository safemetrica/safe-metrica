## 1. 제목

# 위험성평가 공유·근로자 참여 기록을 QR로 시작하세요

## 2. 한 줄 소개

`SafeMetrica Risk Share Link`는 위험성평가 내용을 근로자가 QR로 확인하고, 의견제출과 관리자 검토 기록까지 남길 수 있는 진입형 QR 상품입니다.

## 3. 이런 사업장에 필요합니다

위험성평가표는 있지만 실제 현장 운영기록이 흩어져 있는 사업장에 적합합니다.

```text
위험성평가표는 사무실 파일에 있음
근로자 확인은 종이서명부로 관리
의견제출은 구두, 카카오톡, 메모로 분산
사진은 담당자 휴대폰에 저장
조치 기록은 엑셀 또는 개별 메모로 관리
대표나 관리자가 월간 현황을 한눈에 보기 어려움
```

Risk Share Link는 이 흩어진 기록을 QR 기반으로 연결합니다.

## 4. 핵심 문제

위험성평가를 작성하는 것과 근로자가 실제로 공유받고 의견을 낼 수 있었는지는 별개의 문제입니다.

현장에서 자주 생기는 질문은 다음과 같습니다.

```text
근로자가 위험요인을 확인했는가?
근로자가 안전조치 내용을 확인했는가?
의견이나 아차사고를 제보할 통로가 있었는가?
관리자가 제보를 검토하고 조치 기록을 남겼는가?
월간 단위로 공유확인과 위험제보를 구분해 볼 수 있는가?
```

## 5. SafeMetrica Risk Share Link 해결 방식

근로자는 QR 또는 링크로 접속합니다.

```text
QR 접속
→ 오늘 위험요인 확인
→ 위험성평가 공유 확인
→ 안전조치 주지 확인
→ 의견 없음 또는 위험제보 제출
→ 필요 시 사진 첨부
→ 관리자 접수함 반영
```

관리자는 접수함에서 기록을 분리해 확인합니다.

```text
공유확인
위험제보
아차사고
개선제안
기타
```

처리상태도 별도로 관리합니다.

```text
접수
검토중
조치필요
조치완료
반려
```

## 6. 제공 기능

| 구분      | 제공 내용                            |
| ------- | -------------------------------- |
| 근로자 QR  | 앱 설치 없이 QR 또는 링크 접속 후보           |
| 위험요인 확인 | 오늘 확인할 핵심 위험요인 표시                |
| 공유확인    | 위험성평가 공유 확인 기록                   |
| 안전조치 확인 | 안전조치 주지 확인 기록                    |
| 의견제출    | 위험제보, 아차사고, 개선제안, 기타             |
| 사진첨부    | 위험장소, 조치 전 상태, 개선 필요 위치 사진 첨부 후보 |
| 관리자 접수함 | 공유확인과 검토대상 분리                    |
| 처리상태    | 접수, 검토중, 조치필요, 조치완료, 반려          |
| 월간 요약   | 공유확인, 위험제보, 조치상태 요약 후보           |
| Export  | 고객용 CSV/Excel/PDF 확장 후보          |

## 7. 기대 효과

Risk Share Link는 안전관리 업무를 대신 수행하는 서비스가 아니라, 현장의 공유·확인·의견제출·관리자 검토 과정을 기록으로 남기는 운영지원 도구입니다.

```text
근로자 공유확인 기록 정리
위험제보와 공유확인 분리
관리자 조치상태 관리
사진첨부 기록 보관
월간 요약자료 후보 생성
전체 SafeMetrica 앱 도입 전 가벼운 시작 가능
```

## 8. 가격 후보

가격은 사업장 규모, 현장 수, QR 구성, 보고서 범위, Export 범위에 따라 달라질 수 있습니다.

내부 검토 기준은 다음과 같습니다.

| 구분       | 가격 후보                |
| -------- | -------------------- |
| 기본 과금    | 근로자 1인당 월 2,000원 후보  |
| 월 최소 이용료 | 월 200,000원 후보        |
| 초기 세팅비   | 300,000원~500,000원 후보 |
| 100명 사업장 | 월 200,000원 후보        |
| 200명 사업장 | 월 400,000원 후보        |
| 300명 사업장 | 월 600,000원 후보        |

주의:

```text
위 금액은 내부 기준 후보이며, 실제 견적은 근로자 수, 현장 수, QR 구성, 운영지원 범위에 따라 조정됩니다.
VAT 포함/별도 여부는 견적서에서 명확히 표기합니다.
```

## 9. 전체 SafeMetrica 앱과의 차이

Risk Share Link는 전체 SafeMetrica 앱이 아니라 진입형 QR 상품입니다.

| 구분    | Risk Share Link  | SafeMetrica 정식앱           |
| ----- | ---------------- | ------------------------- |
| 목적    | 위험성평가 공유·참여 기록   | 안전운영 기록 통합                |
| 주요 기능 | QR 확인, 의견제출, 접수함 | TBM, PTW, EB, 현장비서, 월간보고서 |
| 도입 부담 | 낮음               | 중간 이상                     |
| 추천 상황 | 처음 시작            | 전체 운영관리 필요                |
| 확장    | 가능               | 기본 포함                     |

## 10. 현장 적용 예시

```text
50인 이상 제조업 사업장

1. 현장 게시판에 QR 부착
2. 근로자가 작업 전 위험요인 확인
3. 위험성평가 공유 확인 제출
4. 이상 없으면 의견 없음 제출
5. 위험 발견 시 사진과 함께 위험제보 제출
6. 관리자가 접수함에서 검토
7. 조치 필요 항목은 처리상태와 조치 메모 기록
8. 월간 단위로 공유확인·위험제보·조치현황 요약
```

## 11. 중요 안내

Risk Share Link는 법적 면책이나 과태료 방지를 보장하지 않습니다.

```text
본 기능은 사업장의 위험성평가 공유, 근로자 확인, 의견제출, 관리자 검토 과정을 기록으로 남기기 위한 운영지원 기능입니다.
위험성평가의 최종 실시, 위험성 결정, 개선대책 확정, 현장 조치 실행 및 법적 책임은 사업주에게 있습니다.
```

```text
QR 제출만으로 법적 의무 완료
```

```text
관리자가 검토하고 조치 기록을 남겼는지를 QR로 연결하자는 제안입니다.
```
