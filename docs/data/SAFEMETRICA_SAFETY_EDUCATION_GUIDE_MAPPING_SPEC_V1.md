# SafeMetrica Safety Education Guide Mapping Spec v1

## 1. 목적

이 문서는 고용노동부 「2025 안전보건교육 안내서」를 SafeMetrica 앱, Notion RAG 자료 라이브러리, 월간보고서, 고객용 Export, TBM 교육기록 구조에 어떻게 매핑할지 정의한다.

SafeMetrica는 교육 이수 여부를 법적으로 확정하거나 과태료 방지를 보장하지 않는다. 앱은 교육기록 후보, 누락 후보, 증빙 후보, 월간 운영자료를 제공한다.

AI는 후보 제안자이며, 최종 판단과 책임은 관리자와 사업주에게 있다.

| 항목 | 내용 |
| --- | --- |
| 자료명 | 2025 안전보건교육 안내서 |
| 발간 | 고용노동부, 2025.6 |
| 기준 | 2025.6.1 시행 중인 산업안전보건법령 및 안전보건교육규정 |
| 주요 범위 | 근로자 안전보건교육, 관리감독자교육, 채용 시 교육, 작업내용 변경 시 교육, 특별교육, 직무교육, 자체교육, 교육일지, TBM 교육시간 인정 |

## 3. SafeMetrica 반영 원칙

1. 교육 이수 완료를 앱이 확정하지 않는다.
2. 교육시간 충족 여부는 후보값으로만 표시한다.
3. 교육 대상 여부는 업종코드, 상시근로자 수, 근로형태, 사무직/비사무직 구분, 작업유형 확인 후 판단한다.
4. TBM 기록은 정기교육 시간 인정 후보로만 집계한다.
5. 사진, 녹음, 동영상, 앱 기록은 교육 증빙 후보로 분류한다.
6. 고객용 문구에는 과태료 방지, 법적 면책, 교육 인정 보장 표현을 사용하지 않는다.

## 5. 교육시간 매핑 후보

### 5-1. 근로자 정기교육

| 대상 | 기준 교육시간 후보 |
| --- | --- |
| 사무직 종사 근로자 | 매반기 6시간 이상 |
| 판매업무 직접 종사 근로자 | 매반기 6시간 이상 |
| 그 외 근로자 | 매반기 12시간 이상 |

무재해 사업장 또는 일부 업종은 감면 후보가 있을 수 있으나, SafeMetrica는 이를 확정하지 않는다. 산재율 확인서, 업종, 근로자 구분 확인 후 관리자 검토가 필요하다.

### 5-2. 관리감독자 정기교육

| 대상 | 기준 교육시간 후보 |
| --- | --- |
| 관리감독자 | 연간 16시간 이상 |

관리감독자 교육은 사업장 상황, 무재해 여부, 다른 법령 교육 이수 여부 등에 따라 감면 후보가 있을 수 있다. 앱은 후보로만 표시한다.

### 5-3. 채용 시 교육

| 대상 | 기준 교육시간 후보 |
| --- | --- |
| 일용근로자 및 1주일 이하 기간제근로자 | 1시간 이상 |
| 1주일 초과 1개월 이하 기간제근로자 | 4시간 이상 |
| 그 외 근로자 | 8시간 이상 |

### 5-4. 작업내용 변경 시 교육

| 대상 | 기준 교육시간 후보 |
| --- | --- |
| 일용근로자 및 1주일 이하 기간제근로자 | 1시간 이상 |
| 그 외 근로자 | 2시간 이상 |

### 5-5. 특별교육

| 대상 | 기준 교육시간 후보 |
| --- | --- |
| 일반 특별교육 대상 근로자 | 16시간 이상 |
| 단기간 또는 간헐적 작업 | 2시간 이상 |
| 일부 일용근로자 | 2시간 이상 |
| 타워크레인 신호작업 일용근로자 등 | 8시간 이상 |

특별교육은 유해·위험작업 39종과 연결하되, v1에서는 전체 자동 판정하지 않는다.

## 6. TBM 정기교육 시간 인정 매핑

안내서 기준상 작업 전 안전점검회의(TBM)가 현행 규정 및 TBM 가이드라인에 부합하는 형태로 내실 있게 이루어지는 경우 안전보건교육 이수시간으로 인정될 수 있다.

SafeMetrica TBM 기록은 아래 항목을 교육시간 인정 후보 데이터로 저장한다.

| 세메 TBM 필드 후보 | 교육 인정 매핑 |
| --- | --- |
| 작업일자 | 교육일자 |
| 시작시간 | 교육 시작시간 |
| 종료시간 | 교육 종료시간 |
| 작업명 | 교육 주제 |
| 작업유형 | 교육 대상 작업 |
| 주요 위험요인 | 교육내용 후보 |
| 참석자 또는 참석 인원 | 교육대상/참석인원 |
| 관리자/강사 | 주관자/강사 후보 |
| 사진 | 증빙자료 후보 |
| 녹음/음성기록 | 증빙자료 후보 |
| TBM 내용 | 교육일지 후보 |
| 위험성평가 연결 | 교육 주제 근거 후보 |

## 7. TBM 교육시간 합산 후보

SafeMetrica는 TBM 시작시간과 종료시간을 기준으로 반기 또는 월간 누적 시간을 계산할 수 있다.

| 계산 항목 | 설명 |
| --- | --- |
| TBM 1건 시간 | 종료시간 - 시작시간 |
| 월간 합산 | 해당 월 TBM 시간 합계 |
| 반기 합산 | 근로자별 또는 TBM 단위별 합산 후보 |
| 결원 반영 | v1에서는 자동 차감하지 않음 |
| 인정 후보 | 관리자 검토 전까지 “교육시간 후보”로 표시 |

v1에서는 개인별 완전 출결 계산보다 TBM 단위별 교육시간 후보 집계를 우선한다.

## 8. 앱·사진·녹음·동영상 증빙 매핑

안내서에는 TBM 기록과 관련하여 작업일지, 자체 개발 앱, 사진 업로드, 녹음, 동영상 등 다양한 방식의 증빙 가능성이 제시되어 있다.

SafeMetrica 증빙 분류 후보:

| 증빙유형 | SafeMetrica 매핑 |
| --- | --- |
| 앱 기록 | TBM 작성 기록, 작성자, 작성일시 |
| 사진 | 참석/서명, 작업 전 현장, 작업 대상, 조치사진 |
| 녹음 | TBM 음성작성 원천기록 후보 |
| 동영상 | 후속 Evidence v2 후보 |
| 교육자료 | TBM 주제, 위험성평가표, 공단 자료 링크 |
| 교육일지 | 월간보고서 및 고객 Export 출력 후보 |

## 9. 월간보고서 교육 실적 매핑

월간보고서에는 아래 항목을 교육 실적 후보로 표시할 수 있다.

| 항목 | 설명 |
| --- | --- |
| TBM 실시 횟수 | 월간 TBM 기록 수 |
| TBM 누적 시간 | 시작/종료시간 기준 합산 후보 |
| 교육 주제 | 주요 위험요인, 작업유형, 안전공지 |
| 참석/증빙 수 | 사진, 음성, 파일 수 |
| 누락 후보 | 시작/종료시간 없음, 참석자 없음, 증빙 없음 |
| 교육시간 인정 후보 | 관리자 검토 필요 |
| 다음 달 보완 | 교육 누락, 증빙 부족, 위험작업 특별교육 검토 |

## 14. v1에서 하지 않는 것

- 교육 이수 완료 자동 확정
- 교육 대상 여부 자동 확정
- 과태료 방지 보장
- 법정교육 대행 표현
- 특별교육 39종 전체 자동 판정
- 개인별 출결 완전 관리
- 모바일 원격교육 플랫폼 기능 구현
- 교육기관 위탁 여부 검증 자동화
- 고객 민감정보와 RAG 원문 혼합

## 15. 금지 표현

- 법정교육 이수 보장
- 과태료 방지
- 법적 면책
- 교육 인정 보장
- 안전보건교육 대행
- 고용노동부 인정 완료
- KOSHA 인정 보장
- AI가 교육 이수 여부를 확정
- 사고 시 문제없음
- 대표 책임 제거

## 16. 권장 표현

- 정부 기준 자료를 참고한 교육기록 후보입니다.
- TBM 기록은 요건 충족 여부에 따라 정기교육 시간 인정 후보로 검토될 수 있습니다.
- 교육시간과 교육대상 여부는 업종, 근로형태, 작업유형 확인 후 관리자 검토가 필요합니다.
- AI는 후보 제안자이며 최종 판단과 책임은 관리자와 사업주에게 있습니다.
- 실제 기록과 처리 상태는 각 메뉴에서 확인해야 합니다.
