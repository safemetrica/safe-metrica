# SafeMetrica Monthly File Cabinet UX v1

기준일: 2026-06-15
작업명: SafeMetrica 월별 보관함 / 파일 기반 신뢰 UX v1

## 1. 배경

현장 담당자 피드백에서 중요한 방향이 확인되었다.

고객은 클라우드형 SaaS와 AI 대시보드만으로는 데이터 소유감과 안정감을 충분히 느끼지 못할 수 있다.
특히 산업안전, 폐기물, 물류, 제조, 공공 대응이 있는 사업장에서는 다음 감성이 중요하다.

- 내 PC 또는 회사 드라이브에 파일이 남아 있는 느낌
- 월별로 정리된 PDF, Excel, 증빙파일
- 근로감독관, 구청, 발주처, 대표 보고 시 즉시 보여줄 수 있는 자료
- 계약 종료 후에도 고객이 확인 가능한 운영기록
- 복잡한 AI 화면보다 익숙한 업무 프로그램형 보관 경험

이번 기준은 SafeMetrica가 AI가 대신 판단하는 앱처럼 보이는 것을 줄이고,
관리자가 확인하고, 파일로 남기고, 월별로 보관하는 안전운영 기록 시스템으로 보이게 하기 위한 UX 기준이다.

## 2. 핵심 결론

SafeMetrica의 내부는 최신 SaaS 구조로 유지한다.

- Supabase 원장
- evidence_items
- worker_share_confirmations
- worker_reports
- worker_representative_confirmations
- risk_share_item_candidates
- monthly report
- customer CSV export
- AI candidate suggestion

그러나 고객 표면 UX는 아래처럼 제공한다.

- 월별 보관함
- PDF 보고서
- Excel 대장
- 증빙목록
- 첨부사진 ZIP
- 생성일시
- 수정일시
- 생성자
- 전달상태

핵심 문장:

AI는 뒤에서 정리하고, 관리자는 앞에서 확인하며, 고객은 파일로 보관한다.

## 3. 제품 정의 보정

기존 표현:

SafeMetrica는 AI 기반 산업안전 운영 SaaS입니다.

보정 표현:

SafeMetrica는 현장 입력을 모아 관리자가 확인하고, 월별 PDF·Excel·증빙파일로 보관할 수 있게 하는 산업안전 운영기록 SaaS입니다.

영업용 한 줄:

앱에서 입력하고, 관리자는 확인하고, 매월 PDF·Excel·증빙파일로 보관합니다.

감독 대응용 설명:

근로자 공유확인, 위험제보, 조치메모, 근로자대표 참여확인, 증빙자료를 월별 운영기록으로 정리해 필요 시 확인자료로 활용할 수 있습니다.

주의:

- 면책 보장 표현 금지
- 과태료 방지 표현 금지
- 무재해 보장 표현 금지
- 법적 완료 확정 표현 금지
- AI 자동 판단·자동 확정 표현 금지

## 6. 고객이 느껴야 하는 감각

제공해야 할 감각:

- 내 자료가 남아 있다.
- 월별로 정리되어 있다.
- 필요하면 바로 받을 수 있다.
- 구독 중에도 확인 가능하다.
- 계약 종료 시에도 전달 가능한 파일이 있다.
- 감독관, 구청, 발주처가 오면 보여줄 자료가 있다.

피해야 할 감각:

- 앱 안에만 갇혀 있다.
- AI가 뭘 판단했는지 모르겠다.
- 내가 관리한 흔적이 없다.
- 구독을 끊으면 자료를 못 볼 것 같다.
- 다운로드가 어렵다.
- 파일명이 정리되어 있지 않다.

## 7. AI 표현 기준

고객 화면에서 AI를 전면에 내세우지 않는다.

변경 기준:

- AI 분석 결과 → 확인 후보
- AI 판단 → 검토 필요 항목
- AI 브리핑 → 오늘 확인할 항목
- AI가 조치완료 판단 → 관리자 확인 전 후보
- AI 보고서 → 월간 운영기록 초안

SafeMetrica의 기준:

AI는 후보 제안자다.
관리자가 확인한다.
파일로 남긴다.
월별로 보관한다.

7. HWPX 생성 가능성 검토

## 12. 고정 원칙

1. SafeMetrica는 파일 다운로드만 하는 도구가 아니다.
2. SafeMetrica는 현장 입력, 관리자 검토, 월간 보고, 파일 보관을 연결하는 운영기록 SaaS다.
3. 내부 데이터 원장은 Supabase-first로 간다.
4. 고객에게는 PDF·Excel·ZIP 등 익숙한 파일 보관 경험을 제공한다.
5. AI는 고객 화면에서 전면 판단자로 보이지 않게 한다.
6. 관리자 확인과 고객 보관 경험을 중심에 둔다.
7. 법적 면책, 과태료 방지, 무재해 보장, 안전관리대행 표현은 사용하지 않는다.
