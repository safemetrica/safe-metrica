# SafeMetrica Risk Share Version Lock Checklist v1

작업명: `feat: add risk share pack version lock checklist v1`

기준일: 2026-06-14

## 1. 목적

Risk Share Pack 신규 고객 활성화 과정에서 공유항목 Builder 이후 고객 확인과 QR 배포 전 필수 상태를 점검하는 Owner 전용 Version Lock / Go-Live 체크리스트를 추가한다.

## 2. v1 범위

이번 v1은 저장 기능 없이 체크리스트와 링크 후보를 제공한다.

포함 항목:

- 위험성평가 source 접수
- 공유항목 Builder 정리
- 고객 공유범위 확인
- 근로자 표시 항목 확인
- 근로자대표 확인 흐름 확인
- Companies DB 등록 / active 확인
- QR 포스터 준비
- 월간요약 / Export 경로 확인

## 3. 운영 기준

Version Lock은 고객 확인이 끝난 공유항목을 운영상 고정하는 절차다.

이 절차는 위험성평가 작성 대행, 법적 의무 완료, 사고 예방 보장, 조치완료 확정을 의미하지 않는다.

## 4. 후속 작업

1. Version Lock 저장
2. Share item 저장
3. 실제 worker risk-summary와 locked share item 연결
4. QR Poster 자동 생성
5. Go-Live Checklist와 Export 연결
