# SafeMetrica Site Manager TBM First Home Priority v1

## 1. 목적

현장관리자 기본 홈의 운영 우선순위를 명확히 정리한다.

현장관리자는 매일 `오늘 TBM 작성`에서 시작한다.
현재 사용 중인 버튼명과 흐름은 유지한다.

현재 유지할 흐름:

- 오늘 TBM 작성
- 말로 TBM 작성
- 오늘 TBM 작성하기
- 작성 내역 확인

이번 문서는 새 기능 추가나 버튼명 변경이 아니라,
현장관리자 홈에서 TBM이 1순위라는 제품 기준을 잠그기 위한 문서다.

## 2. 고정 원칙

기존 TBM 버튼명과 사용 흐름은 유지한다.

현장관리자 홈의 운영 우선순위는 아래 순서로 둔다.

1. TBM
2. 현장 작업 전 확인
3. 위험성평가 공유팩
4. 월간 정리

## 3. 현장관리자 홈 우선순위

### 1순위 — 오늘 TBM 작성

현장관리자는 매일 아래 흐름 중 하나로 TBM을 작성하거나 확인한다.

- 말로 TBM 작성
- 오늘 TBM 작성하기
- 작성 내역 확인

운영 의미:

- 작업 전 위험요인 공유
- 참석 및 교육 기록 확인
- 사진증빙 연결
- 현장 이슈 또는 특이사항 기록
- 월간보고서와 Evidence Book의 기초 데이터 확보

### 2순위 — 현장 작업 전 확인

TBM 이후에는 오늘 작업 전 확인사항을 점검한다.

- 오늘 작업 전 공유사항
- PTW 고위험작업허가
- 날씨·폭염·우천 등 현장 주의사항
- 현장비서 요약
- 사진증빙 보완 필요 여부

### 3순위 — 위험성평가 공유팩

위험성평가 공유팩은 TBM보다 앞에 두지 않는다.

공유팩은 위험성평가 이후의 확인·의견·제보·관리자 검토 흐름을 관리하는 보조 운영 화면이다.

포함 흐름:

- 위험성평가 최종 공유본 확인
- 근로자 QR 확인
- 위험제보·아차사고·개선제안 확인
- 근로자대표 참여확인
- 관리자 검토
- 고객 전달 자료 준비

### 4순위 — 월간 정리

월말 또는 운영 점검 시 아래 항목을 확인한다.

- 월간보고서
- 고객 전달 자료
- Evidence Book 연결 상태
- 현장참여 검토 상태
- 다음 평가주기 보완 후보

## 4. 화면 문구 기준

현장관리자 홈에서 사용할 수 있는 설명 문구:

현장관리자는 매일 오늘 TBM 작성에서 시작합니다.
말로 작성하거나 기존 방식으로 작성하고,
작성 내역을 확인한 뒤 위험성평가 공유확인·근로자 제보·월간 정리로 이어집니다.

위험성평가 공유팩 연결 문구:

위험성평가 공유팩은 근로자에게 공유한 위험요인의 확인·의견·제보 흐름을 관리하는 보조 운영 화면입니다.

## 5. 변경하지 않는 것

이번 기준에서는 아래 항목을 변경하지 않는다.

- `오늘 TBM 작성` 섹션명
- `말로 TBM 작성` 버튼명
- `오늘 TBM 작성하기` 버튼명
- `작성 내역 확인` 버튼명
- 기존 TBM 작성 흐름
- 기존 고객사가 익숙한 홈 사용 흐름

## 6. 금지 표현

아래 표현은 사용하지 않는다.

- TBM 작성만으로 법적 의무 완료
- TBM 제출로 조치 완료 확정
- AI가 교육 이수 또는 조치 완료 판단
- 위험성평가 공유만으로 법적 책임 해결
- 무재해 보장
- 과태료 방지
- 면책 보장

## 7. 후속 개발 기준

후속 UI 개편 시 아래 순서를 따른다.

1. 기존 TBM 버튼명은 유지한다.
2. TBM 영역을 현장관리자 홈 상단에 둔다.
3. 위험성평가 공유팩은 TBM 아래 보조 카드 또는 별도 메뉴로 둔다.
4. Full SafeMetrica 고객과 Risk Share Pack 단독 고객의 노출 범위는 분리한다.
5. Partner Demo에는 실제 고객 데이터 없이 샘플 흐름만 노출한다.
