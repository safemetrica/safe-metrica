# SafeMetrica Richi Food Factory E-Confirmation Module v1

## 1. 목적

이 문서는 리치코리아 1주일 체험판을 위한 SafeMetrica 식품가공 현장 전자확인·피드백 모듈 기준을 정의한다.

리치코리아는 HACCP 식품가공 현장에서 종이와 볼펜 사용이 어렵고, 근로자 확인과 현장 피드백을 빠르게 남기는 것이 핵심이다.

현대호이스트가 외부현장 설치·AS·작업지시·전후사진·완료보고 중심이라면, 리치코리아는 QR 전자확인·현장 의견·관리자 피드백·상부 보고 흐름이 중심이다.

## 2. 제품 포지션

내부 모듈명:

    SafeMetrica Food Factory Electronic Confirmation Module

고객용 표현:

    리치코리아 현장 전자확인·피드백 운영앱

운영 정의:

    QR 기반 전자확인 기록
    현장 불편·개선의견 제출
    관리자 접수함
    상부 보고용 주간 요약

체험판에서는 전자서명이라는 표현을 전면에 내세우지 않는다. 기본 표현은 QR 기반 전자확인 기록으로 제한한다.

## 3. 핵심 문제

- 식품가공/HACCP 현장 특성상 종이·볼펜 반입이 어렵다.
- 근로자 확인/서명 절차가 불편하다.
- 업무 방해 없이 10~30초 안에 확인해야 한다.
- 현장 불편·개선의견을 빠르게 남길 수 있어야 한다.
- 관리자가 접수 후 피드백하고 상부에 요약 보고할 수 있어야 한다.
- 향후 반복 문서, MSDS, 점검문서, IoT 연동 니즈가 있을 수 있다.

## 4. 1주일 체험판 MVP

1주일 체험판 범위는 아래 6개로 제한한다.

1. QR 기반 근로자 전자확인
2. 확인 내용 snapshot 저장
3. 현장 불편·개선의견 빠른 제출
4. 익명 의견 선택
5. 관리자 접수함
6. 상부 보고용 주간 요약 copy

체험판 목표는 정식 HACCP 문서관리 시스템 구축이 아니라, 현장 근로자가 실제로 부담 없이 확인하고 의견을 남길 수 있는지 검증하는 것이다.

## 5. 1차 제외범위

아래 기능은 1주일 체험판에서 제외한다.

- 손가락 서명패드
- 본인인증 연동
- HACCP 전체 문서관리
- MSDS RAG
- 소방/전기 점검 조치 추적
- 가열실 IoT 센서 API 연동
- 정식 PDF 보고서 자동생성
- 다국어 자동 번역

다국어 기능은 후속 후보로 둔다. 1차는 한국어 copy를 기준으로 구현하고, 이후 외국인 근로자 비율과 필요 언어가 확인되면 QR 화면과 전자확인 copy부터 locale layer를 확장한다.

## 6. 다국어 후속 방향

다국어는 법적 효력 보장 기능이 아니라 현장 이해도 개선 기능이다.

후속 후보 언어:

- 영어
- 중국어
- 베트남어
- 기타 리치코리아 현장 근로자 구성에 따라 확인 필요

다국어 확장 우선순위:

1. 근로자 QR 첫 화면 안내문
2. 전자확인 버튼/동의 문구
3. 현장 불편·개선의견 제출 항목
4. 개인정보 안내 요약
5. 제출 완료 화면
6. 관리자 화면의 원문/번역본 병기 여부 검토

주의:

- 자동 번역 정확성 보장 표현 금지
- 법적 효력 보장 표현 금지
- 번역 문구는 고객 확인 후 확정

## 7. 요청자료

리치코리아에 요청할 자료는 다음과 같다.

1. 근로자 서명부 또는 교육 참석 확인 양식
2. 작업 전 위생/안전 체크리스트
3. HACCP 점검표
4. 일일 작업일지 또는 생산/위생 확인 양식
5. 현장 불편/개선 요청 양식
6. 관리자 조치 기록 양식
7. 상부 보고자료 샘플
8. 월별 점검/교육/개선 보고자료
9. 근로자가 휴대폰을 사용할 수 있는 구역과 시간
10. 가열실 온습도계 또는 도입 예정 센서 자료
11. 외국인 근로자 비율과 필요한 언어 후보

개인정보, 직원명, 전화번호, 세부 거래처명, 내부 민감정보는 가리고 받아도 된다.

## 8. 2차 확장 후보

- 손가락 서명패드
- 확인서 PDF
- 전자확인 감사로그 강화
- 반복 문서 템플릿
- MSDS 보관함
- HACCP 체크리스트 보관
- 소방·전기·정기점검 문서 업로드/조치 추적
- R&R / 책임과 권한 확인
- 가열실 온습도·체감온도 센서 연동
- 다국어 근로자 QR 화면

## 9. IoT 연동 전 확인사항

IoT 연동은 1차 체험판 범위에 넣지 않는다. 별도 견적과 기술 확인이 필요하다.

확인 필요 항목:

- 센서 제조사
- API 제공 여부
- 측정 주기
- 검교정 방식
- 데이터 저장 주기
- 알림 기준
- 비용 구조
- 장애 시 수기 기록 대체 방식

## 10. 금지 표현

아래 표현은 사용하지 않는다.

- 법적 면책
- 처벌 방지
- 무재해 보장
- HACCP 인증 보장
- 종이서명 완전 대체 보장
- 전자확인 법적 효력 보장
- 위험성평가 대행
- 안전관리대행
- AI가 법적 판단 또는 조치완료 확정

## 11. 사용 가능한 표현

- QR 기반 전자확인 기록
- 현장 확인·피드백 운영기록
- 근로자 확인 흐름 지원
- 현장 불편·개선의견 접수 지원
- 관리자 검토·피드백 기록
- 상부 보고용 주간 요약
- 체험판 운영 후 정식 범위 협의

## 14. 운영 판단

리치코리아는 SafeMetrica 전체 운영형을 바로 판매하기보다 1주일 체험판을 통해 현장 입력 저항, QR 접근성, 관리자 피드백 흐름을 확인하는 것이 우선이다.

체험판이 통과되면 반복 문서, 감사로그, PDF, MSDS, IoT, 다국어 화면을 정식 커스터마이징 범위로 분리해 견적화한다.
