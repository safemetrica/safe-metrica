# SafeMetrica Hoist Work Order & Field Safety Module v1

## 1. 목적

이 문서는 현대호이스트 설치·AS 작업기록 앱의 1차 모듈 범위와 자료요청 기준을 정의한다.

현대호이스트는 단순 위험성평가 공유팩 고객이 아니다. 핵심은 외부현장 설치·AS 작업을 작업지시, 외주·일당 확인, 작업 전후 사진, 완료보고, 대표 확인까지 하나의 운영기록으로 남기는 것이다.

## 2. 제품 포지션

내부 모듈명:

    SafeMetrica Work Order & Field Safety Module

고객용 표현:

    현대호이스트 설치·AS 작업기록 앱

확장 표현:

    호이스트·크레인 설치/AS 작업운영 SaaS

현대호이스트는 1호 적용사이자 향후 호이스트·크레인 업종 표준화 파트너 후보로 본다.

## 3. 핵심 문제

- 지침서·매뉴얼이 충분히 정리되어 있지 않다.
- 외주·일당 작업자의 작업 전 확인이 불확실하다.
- 설치·AS 현장 기록이 구두, 카톡, 사진첩에 흩어져 있다.
- 작업지시가 구두 또는 카톡 중심으로 남는다.
- 작업 전후 사진과 완료확인이 한 흐름으로 묶이지 않는다.
- 원청 현장 작업환경이 위험할 수 있으나 자체 확인체계가 약하다.
- 사고, 하자, 분쟁이 발생했을 때 당시 작업상황을 설명할 자료가 부족하다.
- 대표가 매번 현장에 가지 않아도 작업상태를 확인할 수 있는 구조가 필요하다.

## 4. 1차 MVP

처음부터 큰 업무관리 시스템으로 만들지 않는다. 1차 MVP는 아래 5개로 제한한다.

1. 작업지시 등록
2. QR 작업자·외주 확인
3. 작업 전 사진 제출
4. 작업 후 사진 제출
5. 완료보고 / 대표 확인

1차 MVP의 목표는 작업이 있었다는 사실, 누가 확인했는지, 현장 상태가 어땠는지, 완료 후 상태가 어땠는지를 빠르게 찾을 수 있게 하는 것이다.

## 5. 1차 제외범위

아래 기능은 1차 MVP에서 제외하거나 후속 범위로 둔다.

- 독립앱 신규 개발
- 전체 ERP 또는 회계 연동
- 재고관리 전체 기능
- 고객 결제/청구 자동화
- 정식 전자계약
- GPS 기반 실시간 직원 추적
- AI가 작업완료 또는 안전조치 완료를 확정하는 기능
- 법적 면책 또는 분쟁 승소 보장 표현

## 6. SafeMetrica Core 재사용 원칙

현대호이스트 전용 독립앱을 새로 만들지 않는다.

구조는 아래 기준으로 잡는다.

    SafeMetrica Core
    + Hoist Work Order Module
    + tenant copy
    + enabledModules 설정

공통 재사용 기능:

- QR 진입
- 작업자 확인
- 현장 의견/메모
- 사진 증빙
- 관리자 접수함
- 대표 확인
- 월별 보관
- 월간보고서 후보
- Export 후보

## 7. 요청자료

현대호이스트에 요청할 자료는 다음과 같다.

1. 작업지시서 또는 작업계획서 샘플
2. 설치·AS 완료보고서 샘플
3. AS 접수/처리 내역 샘플
4. 외주·일당 작업자에게 보내는 카톡/문자 예시
5. 설치 전/후 사진 예시
6. 고장 부위·교체 부품·설치 완료 사진 예시
7. 고객 완료확인서 또는 인수확인서 샘플
8. 견적서·거래명세서·청구서 양식
9. 현장에서 작업자에게 지키게 하고 싶은 안전수칙 또는 작업 전 확인사항
10. 사고·하자·분쟁 때 꼭 남기고 싶은 기록 항목

개인정보, 고객명, 현장주소, 전화번호, 금액, 작업자 개인정보는 가리고 받아도 된다. 실제 양식 형태와 작업 흐름을 확인하는 것이 목적이다.

## 8. 2차 확장 후보

- 외부현장 작업 전 확인서
- 외주·일당 작업자 확인서
- 고객 완료확인
- 작업완료 보고서 PDF
- 월별 작업기록
- 반복 작업문서 템플릿
- 작업계획서 자동화
- AS 처리내역 자동화
- 정비기록지 자동화
- 부품 교체 기록
- 고객사별 설치·AS 이력 조회

## 9. 가격 방향

현대호이스트는 단순 월구독만으로 잡으면 안 된다. 업무진단, 문서표준화, 1차 구축, 월 운영구독을 분리한다.

가격 후보:

- 업무진단·문서표준화: 200만~300만 원
- 1차 구축비: 800만~1,200만 원
- 월 운영구독료: 80만~150만 원
- 추가 문서양식/PDF/Export: 별도 견적

호이스트 모임 26개 업체 확장 조건이 실제로 있다면 1호 적용 조건을 협의할 수 있다.

1호 적용 조건 후보:

- 구축비 일부 조정 가능
- 대신 호이스트 모임 대상 설명 기회 확보
- 업종 표준화 인터뷰 협조
- 데모/레퍼런스 활용 범위 협의

## 10. 금지 표현

아래 표현은 사용하지 않는다.

- 법적 면책
- 처벌 방지

- 사고 방지 보장
- 분쟁 승소 보장
- 하자 책임 면제 보장
- 위험성평가 대행
- 안전관리대행
- AI가 작업완료 또는 안전조치 완료를 확정

## 14. 운영 판단

현대호이스트는 리치코리아보다 모듈 크기가 크다. 따라서 먼저 문서와 자료요청으로 범위를 잠그고, 이후 1차 MVP 구현으로 들어간다.

현대호이스트의 핵심 가치는 작업지시, 외주확인, 전후사진, 완료보고, 대표 확인을 묶어 사고·하자·분쟁 시 꺼내볼 수 있는 작업운영 기록을 만드는 것이다.
