# SafeMetrica Risk Share Link Feature Truth Map v1

기준일: 2026-06-09
문서 위치: `docs/product/SAFEMETRICA_RISK_SHARE_LINK_FEATURE_TRUTH_MAP_V1.md`
작업명: `docs: add risk share link feature truth map v1`

## 1. 목적

이 문서는 `SafeMetrica Risk Share Link` 상품의 기능별 실제 제공 상태를 정리하기 위한 Feature Truth Map이다.

직전 문서에서 Risk Share Link는 다음 2개 기준으로 정리되었다.

```text
1. Product Spec v1
- 상품 범위
- 대상 고객
- 가격 후보
- 제공 범위
- 제외 범위
- 법적 표현 기준

2. QR UX Spec v1
- 근로자 QR 접근 흐름
- 공유확인과 위험제보 분리
- 관리자 접수함 UX
- 대표/고객 요약 UX 후보
- 고객용 Export 연결 후보
```

본 문서는 영업자료, 랜딩페이지, Partner Demo, 정식앱 구현 시 과장 표현을 막기 위해 기능 상태를 아래 5단계로 분류한다.

```text
운영 가능
제한 운영
준비 중
장기 검토
금지 표현
```

## 2. 기준 원칙

Risk Share Link의 모든 기능 설명은 다음 원칙을 따른다.

```text
있는 기능은 있는 그대로 말한다.
제한된 기능은 제한 운영이라고 말한다.
아직 없는 기능은 준비 중 또는 장기 검토로 분리한다.
법적 효과, 면책, 과태료 방지, 교육 인정, KOSHA 인정은 보장하지 않는다.
AI는 후보 제안자이며 최종 판단과 책임은 관리자와 사업주에게 있다.
```

## 8. 금지 표현

Risk Share Link와 관련해 아래 표현은 사용하지 않는다.

```text
위험성평가 법적 완료
근로자 참여 의무 자동 충족
법정교육 인정 완료
교육시간 인정 보장
과태료 방지
중대재해 면책
법적 방어 보장
무재해 보장
KOSHA 인정 보장
안전조치 완료 확정
AI가 조치완료를 판단
사진 업로드로 증빙 완료
QR 제출만으로 법적 의무 완료
```

## 9. 권장 표현

아래 표현은 사용 가능하다.

```text
위험성평가 공유·확인 과정을 기록으로 남김
근로자 의견제출 경로 제공
관리자 검토와 처리상태 기록
공유확인과 위험제보를 분리 집계
월간 운영자료 후보로 활용
사후 설명자료 후보로 활용 가능
고객용 Export 후보로 정리 가능
AI는 후보 제안자이며 최종 판단은 관리자와 사업주가 수행
```

## 10. 영업자료 표현 기준

### 10-1. 사용 가능 문장

```text
Risk Share Link는 위험성평가 내용을 근로자가 QR로 확인하고,
의견제출과 관리자 검토 기록까지 남길 수 있는 진입형 QR 상품입니다.
```

```text
위험성평가를 작성하는 것에서 끝내지 않고,
근로자가 확인하고 의견을 남길 수 있었는지를 기록으로 남깁니다.
```

```text
공유확인은 공유확인으로,
위험제보는 검토·조치 대상으로 분리해 관리합니다.
```

### 10-2. 제한 문구가 필요한 문장

```text
월간보고서와 고객용 Export는 고객사 데이터 구조와 적용 범위에 따라 제공 범위가 달라질 수 있습니다.
```

```text
교육/TBM 기록 후보는 앱이 자동 인정하는 것이 아니라,
사업장 조건과 관련 기준 확인 후 검토할 수 있는 후보 정보입니다.
```

### 10-3. 사용 금지 문장

```text
이 QR만 운영하면 위험성평가 의무가 완료됩니다.
근로자 참여 의무를 자동으로 충족합니다.
과태료를 방지합니다.
중대재해처벌법 대응이 완료됩니다.
교육시간이 자동 인정됩니다.
```

## 13. 가격표 연결 기준

Risk Share Link 가격은 아직 확정값이 아니라 후보로 둔다.

현재 후보:

```text
월 2,000원/인
월 최소요금 20만 원
초기 세팅비 30만~50만 원
```

표현 기준:

```text
인원수, 현장 수, QR 구성, 보고서 범위, Export 범위에 따라 견적이 달라질 수 있습니다.
```

금지 표현:

```text
모든 사업장 동일가
무조건 인당 2,000원
과태료 방지 비용
법적 면책 비용
```

## 14. Supabase / Export 연결 기준

Risk Share Link 데이터는 장기적으로 Supabase 원장과 고객용 Export에 연결한다.

현재 기준:

```text
내부 JSON 백업과 고객용 CSV는 목적을 분리한다.
고객용 Export에는 내부 UUID, raw payload, Notion URL, API Key, 토큰류를 포함하지 않는다.
```

Risk Share Link Export 후보:

```text
공유확인 목록
위험제보 목록
아차사고 목록
개선제안 목록
조치상태 목록
사진증빙 목록
월간 요약
```
