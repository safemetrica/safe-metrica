# SafeMetrica Partner Demo 안내

> **한 줄 정의**
>
> SafeMetrica는 근로자의 확인과 제보, 현장관리자의 조치, 대표의 운영 확인을 하나의 안전운영 기록으로 연결하는 산업안전 운영기록 SaaS입니다.

## 체험 링크

[https://www.safemetrica.com/partner-demo](https://www.safemetrica.com/partner-demo)

## 체험판의 성격

- 실제 고객 데이터가 없는 샘플 체험판입니다.
- Partner Demo는 정식앱의 과장판이 아니라 정식앱 핵심 흐름의 샘플 모드입니다.
- 실제 고객 DB, Notion, Supabase, Owner Console, API, 환경변수와 연결하지 않습니다.
- 체험 중 생성되는 기록은 실제 고객사 운영기록으로 저장되지 않습니다.

## 3분 체험 순서

### 1. 근로자 체험

- 오늘 TBM 확인
- 위험성평가 공유확인
- 위험제보 제출 흐름 확인

### 2. 현장관리자 체험

- 참여현황 확인
- 위험제보 접수
- 조치사진 샘플
- 현장비서 요약 확인

### 3. 대표 체험

- 전체 운영현황 확인
- 위험성평가표 샘플
- 월간보고서 샘플 확인

## 파트너가 봐야 할 확인 포인트

- 근로자 입력이 관리자 조치 화면으로 이어지는지
- 관리자 조치 흐름이 대표 확인 화면으로 이어지는지
- TBM, 위험제보, 조치사진, 월간보고서가 하나의 운영기록 흐름으로 보이는지
- 고객에게 설명하기 쉬운지
- 실제 영업자료나 랜딩페이지에 넣을 수 있는 문구가 있는지

## 외부 설명 시 권장 문구

- “실제 고객 데이터가 없는 샘플 체험판입니다.”
- “근로자 확인, 관리자 조치, 대표 확인 흐름을 3분 정도로 볼 수 있습니다.”
- “AI는 판단 확정자가 아니라 관리자가 확인할 수 있는 후보와 요약을 제안하는 보조 기능입니다.”
- “최종 판단과 운영 책임은 관리자와 사업주에게 있습니다.”

## 외부 설명 시 금지 표현

다음 표현은 영업자료, 랜딩페이지, 구두 설명 등 외부 커뮤니케이션에 사용하지 않습니다.

- 법적 면책
- 처벌 방지
- 완벽 대응
- 자동 확정
- 법적 증빙 완료
- 안전조치 자동 완료
- 위험성평가 대행
- KOSHA 인정 보장
- 사고 자동 출동
- AI가 법적 판단 확정
- 무재해 보장
- 과태료 방지
- 중대재해 면책

법적 적합성이나 면책을 보장하거나 AI가 최종 판단을 확정하는 것으로 오해될 수 있는 표현을 피하고, 실제 제공 범위와 보조 기능의 역할만 설명합니다.

## 파트너 피드백 요청 항목

- 화면 흐름이 직관적인지
- 현장관리자가 이해하기 쉬운지
- 대표가 관심 가질 포인트가 보이는지
- 영업자료에 넣으면 좋을 문구가 있는지
- 실제 고객에게 보여주기 전 보완할 부분이 있는지

## 카톡 공유용 짧은 문구

```text
SafeMetrica 파트너 체험판입니다.

근로자 → 현장관리자 → 대표 흐름을 3분 정도로 확인할 수 있습니다.

https://www.safemetrica.com/partner-demo

보실 때는 ‘현장 영업에서 고객에게 설명하기 쉬운지’, ‘대표가 관심 가질 흐름인지’ 위주로 봐주세요.
```

---

> **공유 전 주의사항**
>
> 실제 고객명, 실제 고객 데이터, 실제 인증정보나 비밀값을 문서 또는 공유 메시지에 포함하지 않습니다. 제품의 법적 적합성, 행정처분 회피 또는 중대재해 책임 면제를 보장하는 표현을 사용하지 않으며, AI를 최종 판단 주체로 설명하지 않습니다.
