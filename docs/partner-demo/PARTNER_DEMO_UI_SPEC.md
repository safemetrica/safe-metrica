# SafeMetrica Partner Demo UI Spec

## 1. 홈 화면 기준

* max-width: 430px
* dark navy 배경
* 모바일 앱 온보딩 구조
* 상단 브랜드:
  * SafeMetrica
  * 산업안전 운영기록 SaaS
* 메인 제목:
  * 파트너 체험판
* 설명:
  * 역할별로 SafeMetrica의 핵심 기능을 직접 체험해보세요.
  * 3분이면 충분합니다.
* 정보칩:
  * 총 체험시간 약 3분
  * 샘플 데이터
* 역할 카드:
  * 근로자 체험
  * 현장관리자 체험
  * 대표 체험
* 카드 구조:
  * 왼쪽 아이콘
  * 중앙 설명
  * 오른쪽 chevron
  * 카드 전체 Link
* 색상:
  * 근로자: teal/emerald
  * 현장관리자: blue
  * 대표: amber
* 하단:
  * 샘플 데이터 및 실제 고객 DB 미연결 안내

## 2. 후속 화면 공통 구조

* 상단바
* 뒤로가기
* 역할 배지
* 진행바
* 본문 카드
* 하단 CTA
* 완료 피드백 카드
* 체험 모드 안내문

## 3. 공통 문구 원칙

* 운영기록
* 공유확인
* 조치이력
* 사진증빙
* 월간 안전운영 보고서
* 샘플 저장 완료
* 실제 고객 DB에는 저장되지 않습니다

## 4. 금지 표현

* 법적 면책
* 처벌 방지
* 완벽 대응
* 자동 확정
* 법적 증빙 완료
* 안전조치 자동 완료
* 위험성평가 대행
